# 🌐 Ayushman Sharma - Interactive Resume

🚀 A fully responsive and interactive online resume built using **HTML**, **CSS (Tailwind)**, and **JavaScript**. Designed to showcase my skills, projects, certifications, and contact information with animations, smooth scrolling, and a dark/light theme toggle. 

---

## ✨ Features

- 🎨 **Modern Design** – Clean UI with Tailwind CSS and responsive layouts for all devices.
- 🌙 **Dark/Light Mode** – Switch themes seamlessly with a toggle.
- 📄 **Downloadable Resume** – One-click download for PDF resume.
- 🚀 **Smooth Scrolling & Animations** – Better user experience with smooth navigation.
- 📱 **Responsive Design** – Works on desktops, tablets, and mobiles.
- 📧 **Contact Form** – Send messages directly from the website using EmailJS.
- 📌 **Hosted Online** – Viewable 24/7 with Netlify hosting.
